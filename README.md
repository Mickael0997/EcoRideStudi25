# EcoRideStudi25

Conception et déploiement d'une application de covoiturage appelée EcoRide.

## Comment installer

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/EcoRideStudi25.git

Importez la base de données ecoride.sql dans votre serveur MySQL.

Configurez la connexion à la base de données dans le fichier database.php.

Démarrez votre serveur web (par exemple, XAMPP) et accédez à l'application via http://localhost/EcoRideStudi25.

Fonctionnalités
Utilisateurs
Inscription et Connexion : Les utilisateurs peuvent s'inscrire et se connecter à l'application.
Covoiturage : Les utilisateurs peuvent créer, rejoindre et gérer des covoiturages.
Avis : Les utilisateurs peuvent laisser des avis sur les covoiturages.
Profil : Les utilisateurs peuvent gérer leurs informations de profil.
Employés
Gestion des Avis : Les employés peuvent valider ou refuser les avis laissés par les utilisateurs.
Gestion des Covoiturages : Les employés peuvent voir les covoiturages qui se sont mal passés.
Administrateurs
Gestion des Comptes : Les administrateurs peuvent créer des comptes employés et suspendre ou réactiver des comptes utilisateurs et employés.
Statistiques : Les administrateurs peuvent visualiser des graphiques sur le nombre de covoiturages par jour et les gains de la plateforme.
Technologies Utilisées
Frontend : HTML, CSS, JavaScript
Backend : PHP
Base de Données : MySQL
Bibliothèques : Chart.js pour les graphiques
